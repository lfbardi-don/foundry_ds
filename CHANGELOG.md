## 0.0.1

Initial release of **Foundry DS**, a production-grade Flutter design system inspired by OpenAI's aesthetic.

### Features
* **Foundations**: Comprehensive token system for Colors, Typography, Spacing, Radius, and Shadows.
* **Primitives**:
  * `FoundryVStack` & `FoundryHStack` for consistent layout.
  * `FoundryGap` for token-based spacing.
  * `FoundryResponsive` for adaptive UIs.
* **Components**:
  * `FoundryButton`: Primary, Secondary, Outline, Ghost, and Destructive variants.
  * `FoundryInput`: Text fields with validation and icons.
  * `FoundryCard`: Elevated and Outlined variants.
  * `FoundrySurface`: Base container with token-aware styling.
  * `FoundryText`: Typography enforcement (Heading, Body, etc.).
  * `FoundryIcon`: Token-sized icon wrapper.
  * `FoundryDivider`: Horizontal and vertical dividers.
* **Patterns**:
  * `FoundryEmptyState`: User-friendly empty state pattern.
  * `FoundryFormSection`: Structured form layout.
* **Theming**:
  * Full Light and Dark mode support.
  * Context extension (`context.foundry`) for easy access.
